----------------------------------------------------------------
  2. DÉGRADÉ SIGNATURE
----------------------------------------------------------------
 
Dégradé violet → cyan
  CSS    : linear-gradient(135deg, #7C3AED 0%, #06B6D4 100%)
  Angle  : 135° (diagonale haut-gauche vers bas-droite)
  Usage  : hero du site, bandeau de titre PPT, verso carte de visite,
           fond des CTA principaux, titres très importants
 
Variante alternative (plus douce)
  CSS    : linear-gradient(135deg, #7C3AED 0%, #EC4899 50%, #06B6D4 100%)
  Usage  : à réserver pour les moments très spéciaux (page d'accueil
           uniquement par exemple)

Lien        : https://fonts.google.com/specimen/Inter

Tailles recommandées :
  Texte courant    : 16px / weight 400
  Texte secondaire : 14px / weight 400
  Petit texte      : 13px / weight 400

Famille     : JetBrains Mono
Lien        : https://fonts.google.com/specimen/JetBrains+Mono
Graisses    : 400 (regular), 500 (medium)
Usage       : chiffres clés, codes, labels techniques type "// SECTION",

----------------------------------------------------------------
  5. RÈGLES D'USAGE
----------------------------------------------------------------
 
À FAIRE :
  - Mode sombre dominant (fond indigo profond) sur le site et les supports
  - Utiliser le dégradé violet/cyan comme fil rouge transversal
  - Réserver le rose magenta aux moments d'emphase forte
  - Garder beaucoup d'espace blanc/sombre autour des éléments (respiration)
  - Animer les modules au survol avec l'effet "néon cyan"
  - Utiliser JetBrains Mono pour les chiffres clés (effet "data")
 
À ÉVITER :
  - Mélanger plus de 3 couleurs vives sur un même écran
  - Utiliser le rose magenta comme couleur de fond
  - Mettre du texte en violet sur fond indigo (contraste insuffisant)
  - Les ombres lourdes ou les effets 3D
  - Les icônes "filled" ou colorées (rester en outline)
  - Title Case ou ALL CAPS sur les titres longs
 
 
----------------------------------------------------------------
  6. CONTRASTES & ACCESSIBILITÉ (WCAG)
----------------------------------------------------------------
 
Combinaisons VALIDÉES (ratio AA minimum 4.5:1) :
  Blanc cassé (#F8FAFC) sur Indigo (#0A0E27)         OK ✓ (16.8:1)
  Blanc cassé (#F8FAFC) sur Violet (#7C3AED)         OK ✓ (5.2:1)
  Indigo (#0A0E27) sur Cyan (#06B6D4)                OK ✓ (7.1:1)
  Indigo (#0A0E27) sur Blanc cassé (#F8FAFC)         OK ✓ (16.8:1)
 
Combinaisons À ÉVITER :
  Cyan (#06B6D4) sur Indigo (#0A0E27)                NON (2.4:1)
    → Utiliser uniquement pour décoration, JAMAIS pour du texte courant
  Gris ardoise (#64748B) sur Indigo (#0A0E27)        NON (3.8:1)
    → OK pour texte décoratif, NON pour texte essentiel
 
 
----------------------------------------------------------------
  7. DÉCLINAISON PAR SUPPORT
----------------------------------------------------------------
 
[ SITE WEB ]
  - Fond principal : indigo profond
  - Sections alternées : indigo + sections blanc cassé pour aérer
  - CTA principaux : dégradé violet→cyan
  - CTA secondaires : bordure cyan, fond transparent
  - Modules au survol : effet néon cyan
 
[ POWERPOINT ]
  - Slide master 1 (titres) : fond indigo + bandeau dégradé en haut
  - Slide master 2 (contenu) : fond blanc cassé pour la lisibilité
  - Couleur d'accent texte : violet électrique
  - Citations / chiffres : JetBrains Mono en cyan ou violet
  - Pas de transitions agressives, privilégier le fondu
 
[ CARTE DE VISITE ]
  Recto :
    - Fond indigo profond
    - Nom en Space Grotesk 18-22pt blanc cassé
    - Fonction en Inter 9-10pt gris clair
    - Petit motif réseau en filigrane (opacité 15%)
    - Trait dégradé violet→cyan sur un bord
  Verso :
    - Fond blanc cassé
    - Coordonnées en Inter 9pt gris ardoise
    - Logo / monogramme en violet électrique
    - QR code vers le site (optionnel)
  Format conseillé : 85 x 55 mm, papier mat 350g minimum
